# 🚀 Coming Soon Page

A simple and responsive **Coming Soon** landing page built using HTML, CSS and JavaScript. This project is designed to inform visitors that a website, product or service is under development and will be launching soon. It provides a clean user interface with a modern design that works across different screen sizes.

## ✨ Features

* 🎨 Clean and modern UI design
* 📱 Fully responsive layout
* ⚡ Lightweight and fast-loading
* 🌐 Cross-browser compatibility
* 🛠️ Easy to customize
* 🚀 Suitable for websites under development
* 💻 Beginner-friendly project structure

## 🛠️ Built With

* **HTML5**
* **CSS3**
* **JavaScript**

Comming-Soon/

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Bineth-Tharana/Comming-Soon.git
```

### 2. Navigate to the Project Folder

```bash
cd Comming-Soon
```

### 3. Open in Browser

Simply open the `index.html` file in your preferred browser.
